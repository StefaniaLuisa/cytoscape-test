## Taxonomy
To keep the experience understandable, every book should have a single primary thematic cluster assignment that determines its constellation placement. Books can have secondary tags, but the primary cluster should be the anchor. This prevents books from feeling “everywhere” and makes the visualization coherent.
We've set a practical cap on secondary tags so metadata stays meaningful rather than noisy. This cap can be revisited as the real Phase 1 book set comes together and natural thematic patterns emerge.

## The Library (Core Component)
Primary audience: People who want to understand what Gambrell is reading and why, including Sally’s personal and professional network, peer foundations, nonprofit leaders, and curious visitors.
The Library landing page exists to orient visitors quickly, invite exploration, and make browsing feel effortless. It needs to answer the most common entry question"What is Gambrell reading?" while communicating what makes Gambrell distinct: curiosity, rigor, and a human-centered point of view.
A realistic success signal for this **page is that visitors explore at least three to four books in a single session**. Where appropriate, the experience can offer passive ways to stay connected, such as checking out the latest Book Club selection or signing up for the newsletter.
The page will include a hero section, an Essentials grouping, thematic clusters based on taxonomy, a Gambrell Galaxy contributions area (initial curated highlights), a Constellation View as the primary interactive experience, and a List View as an accessible alternative.
The constellation will be optimized for desktop. Mobile users will receive a streamlined path centered on list browsing and clear Book Detail pages.

## Book Detail Panel & Post Content Requirements
Each book needs a clear, structured way for visitors to dive deeper. The Book Detail overlay and Book post template provide both a quick‑view option and a full content page.
### Book Detail Panel
Displayed when a user clicks on a book node the the panel will feature:
* Cover image  
* Title  
* Author  
* Primary Category (Gambrell Pillar Tag \- emphasized)  
* Secondary Categories (displayed if present; when present they are grayed out unless hovered)  
* "Learn More" link to full Book post  
* Bookmark icon to add to send list

### Functionality of Panel Interactions
The primary category is emphasized in the overlay. Secondary categories, if present, appear in a grayed-out state unless hovered, at which point they display in the hover color of the linked category. 
If a category name is clicked, the view updates to display that category's books. Clicking the book icon or anywhere outside the categories loads the full [Book Post Popover](https://bookshowcase.gsplugins.com/books-popup/), keeping the visitor on The Library page.
